# 💰 Console Banking System

Консольное банковское приложение на **Java 16+** с использованием **Spring Framework (Spring Context)** для управления пользователями и их счетами.

## 📋 Возможности

- Создание пользователей с автоматическим присвоением ID  
- Создание и закрытие счетов  
- Пополнение, снятие и перевод средств между счетами  
- Просмотр всех пользователей и их счетов  
- Защита от закрытия единственного счета  
- Использование Spring DI для управления зависимостями  

---

## 🧩 Доступные операции

| Команда | Описание | Входные параметры |
|---------|----------|-------------------|
| `USER_CREATE` | Создать нового пользователя + первый счёт | Логин |
| `SHOW_ALL_USERS` | Показать всех пользователей и их счета | Нет |
| `ACCOUNT_CREATE` | Создать новый счёт для существующего пользователя | ID пользователя |
| `ACCOUNT_CLOSE` | Закрыть счёт (остаток → первый счёт пользователя) | ID счёта |
| `ACCOUNT_DEPOSIT` | Пополнить счёт | ID счёта, сумма |
| `ACCOUNT_WITHDRAW` | Снять деньги со счёта | ID счёта, сумма |
| `ACCOUNT_TRANSFER` | Перевести деньги между счетами | ID отправителя, ID получателя, сумма |

## 🚀 Запуск

### Требования
- Java 16 или выше  
- Maven 3.6+  


### Запуск через Maven (рекомендуемый)

```bash
# 1. Клонируй репозиторий
git clone https://github.com/Ya-slv/spring-core-bank-app.git
cd spring-core-bank-app

# 2. Собери проект
mvn clean compile

# 3. Запусти приложение
mvn exec:java -Dexec.mainClass="org.example.Main"
